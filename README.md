# Zhuqi Li — Personal Website

Professional personal site for [Zhuqi Li](https://github.com/JeffryLee), Member of Technical Staff at xAI.

**Live site:** https://jeffrylee.github.io/

## Local preview

Open `index.html` in a browser, or serve locally:

```bash
# Python
python3 -m http.server 8000

# or Node
npx serve .
```

Then visit http://localhost:8000

## Deploy (GitHub Pages)

This repository is configured for **GitHub Pages** from the `main` branch root.

1. Push to `main`
2. In the repo: **Settings → Pages → Source: Deploy from a branch → `main` / `/ (root)`**
3. Site will be available at `https://jeffrylee.github.io/`

For a custom domain, add a `CNAME` file and configure DNS.
